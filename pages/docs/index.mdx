## Getting Started

1. Open the terminal and run the following commands:

    ```bash
    git clone https://github.com/Hai-Lucy/lucy
    cd lucy
    npm install
    ```

2. Wait for all the dependencies to be installed.
3. Rename `.env.example` to `.env` and fill in the values.
4. Optionally edit `config.js`.
5. Type `npm run start` to start the bot.

If you need any additional help, make sure to read our guides [here](docs/commands/installation.md).
